# ApexLegendsアプリのテスト仕様書

## Legend検索機能
### 検索画面（トップ画面）
#### 正常系 検索画面表示
 - 検索バーにlocalhost:8080/topと入力  
   * [ ] 検索画面が表示されること  

#### 異常系 検索画面表示
 - 検索バーにlocalhost:8080/tapと入力  
   * [ ] エラー画面に遷移すること（404 Not Found）  

### 全件検索
#### 正常系 全件検索
 - 検索画面のテキストボックスに何も入力しない状態で「検索」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] レジェンドがリストで全件表示されること  

### 部分一致検索
#### 正常系 部分一致検索
 - 検索画面のテキストボックスに「ア」と入力し、「検索」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] 名前に「ア」の入ったレジェンドがリストで表示されること  
   - 備考
      - 小さい「ア」の入ったレジェンドも表示される  

 - 検索画面のテキストボックスに「ハ」と入力し、「検索」ボタンをクリック  
   * [ ] エラーが表示されないこと
   * [ ] 名前に「ハ」の入ったレジェンドがリストで表示されること  

   - 備考
      - 濁音「バ」の入ったレジェンドも表示される
      - 半濁音「パ」の入ったレジェンドも表示される  

 - 検索画面のテキストボックスに「ﾊ」と入力し、「検索」ボタンをクリック  
   * [ ] エラーが表示されないこと
   * [ ] 名前に「ハ」の入ったレジェンドがリストで表示されること  

   - 備考
      - 濁音「バ」の入ったレジェンドも表示される
      - 半濁音「パ」の入ったレジェンドも表示される  

 - 検索画面のテキストボックスに「い」と入力し、「検索」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] 名前に「い」の入ったレジェンドがリストで表示されること  

   - 備考
      - 小さい「イ」の入ったレジェンドも表示される  

 - 検索画面のテキストボックスに「a」と入力し、「検索」ボタンをクリック
   * [ ] リストが表示されないこと  

 - 検索画面のテキストボックスに「1」と入力し、「検索」ボタンをクリック
   * [ ] リストが表示されないこと  

## Legend詳細画面表示機能
### 詳細画面表示
#### 正常系 詳細画面表示
 - ブラッドハウンドの「詳細」ボタンをクリック
   * [ ] エラーが表示されないこと
    * [ ] レジェンド詳細画面に遷移し、ブラッドハウンドの情報が表示されること  
 
 - 「トップ画面に戻る」リンクをクリック
   * [ ] エラーが表示されないこと
   * [ ] 検索画面（トップ画面）に遷移すること  

## レジェンドのレビュー機能
### レビュー一覧を表示 
#### 正常系 レビュー一覧を表示 
 - レジェンド検索で表示されたリストのジブラルタルの「一覧」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] ジブラルタルのレビュー一覧画面に遷移すること  
   * [ ] レジェンド名にジブラルタルのID（2）と名前(ジブラルタル)が表示されていること

 - 「トップページに戻る」リンクをクリック
   * [ ] エラーが表示されないこと
   * [ ] 検索画面（トップ画面）に遷移すること  

### レビューの編集
#### 正常系 レビューの編集
 - ライフラインのレビュー一覧からレビューID2番の「編集」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] レビューID2番の編集画面に遷移すること
   * [ ] 現在登録されているレビュー内容が初期値として設定されていること  

 - 利用者名を「takahashi→sato」、年齢を「21→空白」、評価を「5→7」、一言コメントを「使ってて楽しかた→使ってて楽しかった」に変更し、「更新」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] レビュー編集確認画面に遷移し,上記で入力されたものが表示されること  

 - レビュー編集確認画面で「もどる」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] レビュー編集画面に戻り、上記で入力したものが初期値として入力されていること  

 - 一言コメントを「」（空白）に変更し、「更新」ボタンをクリック
 - レビュー編集確認画面に遷移し、「更新実行」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] レビュー編集確認画面に遷移し,上記で入力されたものが表示されること  

 - レビュー編集画面で「更新」ボタンをクリック
 - レビュー編集確認画面に遷移し、「更新実行」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] 完了画面に遷移すること  

 - レビュー編集画面で「もどる」をクリック
   * [ ] エラーが表示されないこと
   * [ ] レビュー一覧画面に戻ること  
 
 - レビュー編集画面の「トップ画面に戻る」リンクをクリック
   * [ ] エラーが表示されないこと
   * [ ] 検索画面（トップ画面）に遷移すること  

 - 完了画面の「トップページに戻る」リンクをクリック
   * [ ] エラーが表示されないこと
   * [ ] 検索画面（トップ画面）に遷移すること  

  #### 異常系 レビューの編集
 - ライフラインのレビュー一覧からレビューID2番の「編集」ボタンをクリック
 - 利用者名を「takahashi→空白」、年齢を「-1」、プレイ日を「2025/03/03→未指定」、評価を「選択してください。」、一言コメントを「1234・・・・130」に変更し、「更新」ボタンをクリック
   * [ ] エラーメッセージ表示：利用者名の下に「お名前を入力してください。」
   * [ ] エラーメッセージ表示：年齢の下に「正の整数で入力してください。」
   * [ ] エラーメッセージ表示：プレイ日の下に「日付を入力してください。」
   * [ ] エラーメッセージ表示：評価の下に「1-10で指定してください。」
   * [ ] エラーメッセージ表示：一言コメントの下に「128文字以内で入力してください。」  

 - ライフラインのレビュー一覧からレビューID2番の「編集」ボタンをクリック
 - 年齢を「空白→にじゅういっさい」、プレイ日を「2025/03/06→2028/04/01(未来日)」、「更新」ボタンをクリック
   * [ ] エラーメッセージ表示：プレイ日の下に「指定の形式で入力してください。」
   * [ ] エラーメッセージ表示：プレイ日の下に「今日以前の日付を入力してください。」  

### レビューの削除
#### 正常系 レビューの削除
 - ミラージュのレビュー一覧からレビューID5番の「削除」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] レビューID5番の削除確認画面に遷移すること
   * [ ] レビューID5番の現在の登録内容が表示されていること  
 
 - レビュー削除確認画面で「もどる」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] レビュー一覧画面に戻ること  

 - レビュー一覧からレビューID5番の「削除」ボタンをクリック
 - レビュー編集確認画面に遷移し、「削除実行」ボタンをクリック
   * [ ] エラーが表示されないこと
   * [ ] レビュー削除完了画面に遷移すること
