# ApexLegendsアプリのテスト仕様書

## レジェンド検索機能
### 検索画面（トップ画面）
#### 正常系 検索画面表示
 - 検索バーにlocalhost:8080/topと入力  
   * [x] 検索画面が表示されること  

#### 異常系 検索画面表示
 - 検索バーにlocalhost:8080/tapと入力  
   * [x] エラー画面に遷移すること（404 Not Found）  

### 全件検索
#### 正常系 全件検索
 - 検索画面のテキストボックスに何も入力しない状態で「検索」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] レジェンドがリストで全件表示されること  

### 部分一致検索
#### 正常系 部分一致検索
 - 検索画面のテキストボックスに「ア」と入力し、「検索」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] 名前に「ア」の入ったレジェンドがリストで表示されること  
   - 備考
      - 小さい「ア」の入ったレジェンドも表示される  

 - 検索画面のテキストボックスに「ハ」と入力し、「検索」ボタンをクリック  
   * [x] エラーが表示されないこと
   * [x] 名前に「ハ」の入ったレジェンドがリストで表示されること  

   - 備考
      - 濁音「バ」の入ったレジェンドも表示される
      - 半濁音「パ」の入ったレジェンドも表示される  

 - 検索画面のテキストボックスに「ﾊ」と入力し、「検索」ボタンをクリック  
   * [x] エラーが表示されないこと
   * [x] 名前に「ハ」の入ったレジェンドがリストで表示されること  

   - 備考
      - 濁音「バ」の入ったレジェンドも表示される
      - 半濁音「パ」の入ったレジェンドも表示される  

 - 検索画面のテキストボックスに「い」と入力し、「検索」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] 名前に「イ」の入ったレジェンドがリストで表示されること  

   - 備考
      - 小さい「イ」の入ったレジェンドも表示される  

 - 検索画面のテキストボックスに「a」と入力し、「検索」ボタンをクリック
   * [x] リストが表示されないこと  

 - 検索画面のテキストボックスに「1」と入力し、「検索」ボタンをクリック
   * [x] リストが表示されないこと  

## レジェンド詳細画面表示機能
### 詳細画面表示
#### 正常系 詳細画面表示
 - レジェンド検索で表示されたリストのブラッドハウンドの「詳細」ボタンをクリック
   * [x] エラーが表示されないこと
    * [x] レジェンド詳細画面に遷移し、ブラッドハウンドの情報が表示されること  
 
 - 「トップ画面に戻る」リンクをクリック
   * [x] エラーが表示されないこと
   * [x] 検索画面（トップ画面）に遷移すること  

## レジェンドのレビュー機能
### レビュー一覧を表示 
#### 正常系 レビュー一覧を表示 
 - レジェンド検索で表示されたリストのジブラルタルの「一覧」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] ジブラルタルのレビュー一覧画面に遷移すること  
   * [x] レジェンド名にID（2）と名前(ジブラルタル)が表示されていること  

 - レジェンド検索で表示されたリストのオルターの「一覧」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] オルターのレビュー一覧画面に遷移すること  
   * [x] レジェンド名にID（26）と名前(オルター)が表示されていること  
   * [x] （レビューが存在しません）と表示されること  

 - 「トップページに戻る」リンクをクリック
   * [x] エラーが表示されないこと
   * [x] 検索画面（トップ画面）に遷移すること  

### レビューの編集
#### 正常系 レビューの編集
 - ライフラインのレビュー一覧からレビューID2番の「編集」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] レビューID2番の編集画面に遷移すること
   * [x] 現在登録されているレビュー内容が初期値として設定されていること  

 - 利用者名を「takahashi→あいうえおかきくけこ」、年齢を「21→空白」、評価を「5→7」、一言コメントを「使ってて楽しかた→あ×128」に変更し、「更新」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] レビュー編集確認画面に遷移し,上記で入力されたものが表示されること  

 - レビュー編集確認画面で「もどる」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] レビュー編集画面に戻り、上記で入力したものが初期値として入力されていること  

 - 一言コメントを「」（空文字）に変更し、「更新」ボタンをクリック
 - レビュー編集確認画面に遷移し、「更新実行」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] 完了画面に遷移すること  

 - 完了画面の「トップページに戻る」リンクをクリック
   * [x] エラーが表示されないこと
   * [x] 検索画面（トップ画面）に遷移すること  

 - レビュー編集画面で「もどる」をクリック
   * [x] エラーが表示されないこと
   * [x] レビュー一覧画面に戻ること  
 
 - レビュー編集画面の「トップ画面に戻る」リンクをクリック
   * [x] エラーが表示されないこと
   * [x] 検索画面（トップ画面）に遷移すること

### レビューの削除
#### 正常系 レビューの削除
 - ミラージュのレビュー一覧からレビューID5番の「削除」ボタンをクリック
   * [x] エラーが表示されないこと
   * [x] レビューID5番の削除確認画面に遷移すること

   * [x] 完了画面に遷移すること
